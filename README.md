# Exam Study Tracker

A static study-plan webapp for the June 2026 exam schedule.

## Share as a Link

Fastest no-code option:

1. Go to https://app.netlify.com/drop
2. Drag this `studywebapp` folder onto the page.
3. Netlify will give you a public link you can send to your friend.

GitHub Pages option:

1. Create a public GitHub repository.
2. Upload these files to the repository root.
3. In the repo settings, enable Pages from the default branch root.
4. Share the GitHub Pages URL.

The app is fully static. It only needs `index.html`, `styles.css`, `app.js`, `manifest.json`, `sw.js`, `favicon.svg`, and `.nojekyll`.
